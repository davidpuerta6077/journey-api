Módulo de carga masiva de usuarios plataforma

Instrucciones
  1. Descargar
  2. Cargar la plantilla en la opción CARGAR ARCHIVO.
  3. Procesar el archivo con la opción PROCESAR ARCHIVO CARGADO.
  4. Finalizado el proceso pueden surgir dos escenarios:
    • Hay errores: se genera un documento Excel con los registros con inconsistencias.
    • No hay errores: se muestra un mensaje de éxito.

Nota: Para cargar un nuevo archivo, usar la opción

Procesar archivo
